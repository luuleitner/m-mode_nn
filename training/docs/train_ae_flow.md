## Key Decision Points

1. **CLI Arguments**
   - `--dry-run`: Exit after config load
   - `--validate-config`: Validate and exit
   - `--override`: Apply config overrides

2. **Data Loading**
   - Pickle exists: Load cached datasets
   - No pickle: Create from raw data and cache

4. **Training Mode**
   - Restart enabled: Resume from checkpoint
   - Fresh training: Start from scratch

5. **Checkpointing**
   - Best model: When validation improves
   - Latest: Every epoch
   - Restart: Every N epochs
   - Emergency: On interruption
